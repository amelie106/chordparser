Welcome to chordparser's documentation!
=======================================

chordparser is a Python 3 package that provides a musical framework for analysing chords. Click on these links to find more about `what it is <whatiscp.rst>`_ and `why it is <whycp.rst>`_.

Get started with `Installation <installation.rst>`_ and then get an overview with the `Quickstart <quickstart.rst>`_. The rest of the docs describe each musical class that chordparser provides in detail so you can use the framework effectively.
